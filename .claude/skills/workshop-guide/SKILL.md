# Workshop Guide

維護 `WORKSHOP.md` 這份文件，記錄 Claude Code Skills workshop demo 的完整準備流程，讓參與者可以照著重現整個過程。

## 職責

- 更新 `WORKSHOP.md` 的步驟內容
- 有變動時修改對應步驟，不是新增一份文件
- 保持步驟清楚、可重現

## 每個步驟的格式

```markdown
## Step N — 步驟標題

**做了什麼：** 簡短描述

**為什麼：** 決策原因

**怎麼做：**
1. 操作一
2. 操作二

**結果：** 產出或結論
```

## 注意事項

- 步驟順序反映實際執行順序
- 指令要完整可複製貼上
- 決策原因要寫清楚，讓參與者理解為什麼這樣做
- 如果步驟有變動，更新既有步驟而非新增
